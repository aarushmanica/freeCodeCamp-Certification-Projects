Variables

Variables can be referred to as containers to store data

The advantage of creating Python variables is that they do not need to be initialised, meaning they don't need to be declared with a particular data type.

x = 5
y = "Hello World"
print(x)
print(y)

The data type of variables can change even after they have been set

x = 12
x = "John" print(x) # Will return 'John' instead of 12

Object Oriented Programming

OOP is a programming paradigm that uses objects and classes in order to implement real-world entities

Abstraction

Hiding complex internal details and showing only essential features

E.g when using a car, the driver (user) only needs to know how to interact with the interface such as transmission, brakes and accelerator but does not need to know more complex functionality such as how the engine works or temperature control etc.

Inheritance

A new child class derives attributes, properties and methods from its parent class in order to promote code reusability and lessen code redundancy.

E.g A car object can re-use or inherit features of the 'Vehicle' superclass such as wheels, brakes, accelerator or engine without having to make its own implementation

Encapsulation

Bundling data and methods that operate on that data into a single unit i.e a class.

E.g consider a 'Bank' class. The 'balance' attribute is made private meaning external code cannot access or modify it (can't write account.balance = 500000). This can enhance the program security and protect data integrity.

Generalisation

Extracting common features such as attributes, properties and methods from two classes and creating a new general superclass or base class.

E.g the common features of a car and motorcycle object include wheels or transmission type which can form the superclass of 'Vehicle'

Polymorphism

"Many forms", where objects of different classes to be treated as objects of a common superclass.

E.g consider the superclass Animal that has a method animalSound(). The subclasses of Animal could include Dog, Cat or Pig, which would have their own implementation of the animalSound() method such as that the dog woofs, cat meows or the pig oinks.
